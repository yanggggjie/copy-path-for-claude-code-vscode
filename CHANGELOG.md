# Changelog

## [1.1.0] - 2026-03-11

### Added

- Copy file/folder path as `@path` reference
- Copy with line number `@path#Lx`
- Copy with line range `@path#Lx-y`
- Multi-cursor support
- Multi-file explorer selection
- Explorer keyboard shortcut (`Alt+C`) with multi-file support
- Editor and explorer context menu integration
- Configurable trailing space
- Configurable separator (Space/Newline)
- Configurable notification
